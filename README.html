• 间歇式反应釜

  • 间歇式反应釜（简单）文件展示了一个间歇式反应釜的进料和出料
      • 间歇式反应釜（简单一级反应）展示了反应A->B在间歇式反应釜中的进程。最上方流表示物料进釜。然后A物质给予队列（间歇式反应釜多次反应可以视为一个队），A库进料，开始反应，由于反应为简单一级反应，故反应速度正比于A浓度。由图可知，反应为指数型。
      Interface处可以调节反应时间和速率常数。
    不过个人能力有限，没有做出物质必须在出炉后才能开始反应流，也就是前五分钟的数据毫无价值。该模型不含间歇式反应釜的停釜时间。

  • 连续搅拌釜：

  • 仍旧考虑简单一级反应，反应釜具有恒定的进料出料速率且相等。含有固定百分比的物料进入反应釜，然后反应釜内发生反应。旋钮用于控制反应速率。可以看到，不管如何调节反应速率，釜内物料AB浓度最终收敛。

  • 活塞流反应釜：

  • 考虑简单一级反应，物质A在活塞中发生反应，且不存在返混现象。传送带表征活塞推动，该案例为了模拟极慢的推动速度，取10000倍k（反应速率）。案例不能展示活塞流，不过将数个传送带串联即可。可以看到，输出存在震荡，将速率常数调高应该可以较为真实的模拟活塞流。
